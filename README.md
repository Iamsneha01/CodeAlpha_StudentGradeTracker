# Student Grade Tracker

A simple Java-based application to manage and analyze student grades. This program allows users to input student details, calculate statistical insights, and generate a summary report through a console-based interface.

---

##  Features

-  Add student names and grades
-  Calculate:
  - Average score
  - Highest score
  - Lowest score
-  Store student data using `ArrayList`
-  Display a detailed summary report of all students
-  Menu-driven console interface
-  Easy to use and extend

---

##  Technologies Used

| Technology | Description |
|------------|-------------|
| Java | Core programming language |
| ArrayList | Dynamic data storage |
| OOP | Object-Oriented Programming principles |
| Scanner | User input handling |
